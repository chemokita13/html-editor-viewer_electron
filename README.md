## Instalation:
- With the windows installer -> execute instaler.exe
- Creating your own installer:
```shell
$ git clone https://github.com/chemokita13/html-editor-viewer_electron
$ cd html-editor-viewer_electron
$ npm install
$ npm start-forge
$ npm package
$ cd out
```
And run the .exe file
